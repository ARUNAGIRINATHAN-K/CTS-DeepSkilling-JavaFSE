# Spring Core - Basic Container Routing

An initial demonstration of Spring's Inversion of Control (IoC) mechanics alongside classical XML-based Application Context routing natively resolving dependency paths.

## Highlights
* **Dependency Injection:** Designed internal configurations explicitly setting `BookRepository` parameters deep inside `BookService` leveraging clean setter patterns seamlessly resolving dependency expectations natively.
* **Component Routing:** Provisioned straightforward execution architectures defining strict mapping behaviors solely utilizing `<bean>` XML references inside the core `applicationContext.xml`.
* **Application Context:** Evaluated the classical runtime container natively retrieving pre-wired backend objects smoothly querying explicit `ClassPathXmlApplicationContext` boundaries safely.
