## Course Modules

### **Week 1: Core Java & Design Foundations**

#### Design Patterns
- **[Singleton Pattern](./Week%201/Design%20Pattern/SingletonPatternExample/)**
  - Implements the Singleton creational pattern ensuring single instance management
  - Use case: Database connections, logging frameworks, configuration managers
  - Thread-safe implementations with eager and lazy initialization approaches

- **[Factory Pattern](./Week%201/Design%20Pattern/FactoryMethodPatternExample/)**
  - Demonstrates Factory Method pattern for decoupled object creation
  - Abstracts object instantiation logic from client code
  - Enables flexible dependency resolution and polymorphic object generation

#### Data Structures & Algorithms
- **[Data Structures & Algorithms](./Week%201/Data%20Structure/)**
  - Core algorithmic problem-solving exercises
  - Practical implementations of arrays, linked lists, trees, and graphs
  - Time and space complexity analysis for optimal solution selection
  - Real-world problem scenarios requiring algorithmic thinking

#### Database Management
- **[PL/SQL](./Week%201/PLSQL/)**
  - Backend database development with procedural SQL
  - Custom stored procedures, functions, and triggers
  - Relational database design and optimization techniques
  - Transaction management and query performance tuning

#### Testing Frameworks
- **[JUnit Testing Exercises](./Week%201/J-Unit/)**
  - Test execution lifecycle and assertion methodologies
  - JUnit 4 framework fundamentals and test organization
  - Test fixtures, parameterized testing, and test suites
  - Best practices for unit test design and coverage

- **[Mockito Integration Exercises](./Week%201/Mockito/)**
  - Mock object creation and behavior simulation
  - Stubbing complex dependencies and verifying interactions
  - JUnit 5 integration with advanced mocking scenarios
  - Test isolation and dependency management patterns

#### Enterprise Logging
- **[SLF4J & Logback Configuration](./Week%201/SLF4J/)**
  - Enterprise-grade logging configuration and management
  - SLF4J abstraction layer with Logback implementation
  - Standardized console output formatting and log levels
  - Appenders, layouts, and conditional logging strategies
