# Financial Forecasting Model 

A straightforward, recursive Java implementation designed to predict future financial values based on historical growth rates. 

## Overview
This repository contains a simple financial forecaster that leverages recursion to project future values. It demonstrates fundamental computer science principles (recursion, base cases, algorithmic analysis) applied to a real-world financial calculation.

## How to Run
1. Make sure you have the Java Development Kit (JDK) installed.
2. Compile the source code:
   ```bash
   javac src/*.java
   ```
3. Run the application:
   ```bash
   java -cp src Main
   ```

## Project Structure
- `src/FinancialForecasting.java`: Contains the core recursive logic for projecting future values.
- `src/Main.java`: The main execution script with sample inputs to demonstrate the forecasting.
- `Analysis.md`: A concise theoretical breakdown covering recursive algorithms, time complexity, and strategies for optimization (like memoization or direct mathematical computation).
