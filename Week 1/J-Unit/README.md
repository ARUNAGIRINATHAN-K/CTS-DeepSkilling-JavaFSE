# JUnit Testing Exercises

A foundational overview of Unit Testing in Java leveraging JUnit 4 within a Maven architecture. 

## Highlights
* **Project Initialization:** Configured basic Maven project constraints encompassing standard JUnit validation.
* **Basic Assessments:** Setup a straightforward `Calculator` evaluating primitive method calls.
* **Assertion Testing:** Validated standard logic flows utilizing explicit JUnit assertions (`assertEquals`, `assertTrue`, `assertNull`, etc).
* **AAA Architecture:** Organized robust test suites applying the modern Arrange-Act-Assert testing pattern alongside testing fixtures and lifecycle hooks (`@Before` & `@After`) effectively.
